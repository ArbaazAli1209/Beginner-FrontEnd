# 📚 StudySync - Responsive Educational Landing Page

StudySync is a modern, responsive landing page for an online learning platform. It is built using **HTML5** and **CSS3**, focusing on clean UI, responsive layouts, and modern web design principles.

## 🌐 Live Demo

```
https://studysync-project-nu.vercel.app/
```

---

## 📖 About

StudySync is a front-end project that showcases the homepage of an online education platform. The website is designed to provide an engaging user experience with responsive layouts, attractive sections, smooth animations, and modern styling.

This project was created to practice responsive web design, Flexbox, CSS Grid, animations, and UI development using only HTML and CSS.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern & Clean User Interface
- 🏠 Hero Section with Call-to-Action Buttons
- 🧭 Responsive Navigation Bar
- 🏢 Trusted Companies Section
- 🚀 Features & Benefits Cards
- 💬 Testimonials Section
- 📧 Newsletter Subscription Form
- 🔗 Multi-column Footer
- ⚡ Smooth CSS Animations
- 📐 Flexbox & CSS Grid Layouts
